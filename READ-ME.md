## My Topic
> **09: Why Hydroponics?" Animated Infographic**
> Build a one-page scrolling infographic that sells the benefits of hydroponics. Include 4 big animated stat counters (for example: 90% less water, 3x faster growth, 0 pesticides, 365 days a year) that count up when the page loads, each with an icon and caption. Add smooth fade-in sections as the user scrolls down. End with a "Join Farmspherica" call-to-action. Make it bold and modern with subtle green gradient touches and confident typography.

## ✨ Features Implemented
* **Animated Stat Counters:** Dynamic numbers that count up upon page load to highlight key hydroponics data.

* **Scroll Reveal Animations:** Smooth fade-in effects as the user scrolls down the page to keep the experience engaging.

* **Modern UI/UX:** A bold, clean design featuring confident typography and subtle green gradients to match the agricultural theme, with a built in button to switch from light mode to dark mode

* **Engaging CTA:** Ends with a clear "Join Farmspherica" call-to-action. 

* **Tech Stack:** Pure HTML, CSS, and JavaScript.
